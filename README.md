# AlertBukavu

## Installation

```bash
npm install
npm start
```

Ouvre http://localhost:3000 dans ton navigateur.

## Sans Node.js

Tu peux aussi ouvrir directement `public/login.html` dans ton navigateur — ça marche sans serveur.

## Firebase

La configuration Firebase est déjà incluse dans `public/firebase-config.js`.

Active dans la console Firebase :
- Firestore Database (mode test)
- Authentication (Email/Password)

## Pages

- `login.html` — Connexion
- `register.html` — Inscription  
- `index.html` — Fil d'alertes (dashboard)
- `publier.html` — Publier une alerte
- `stats.html` — Statistiques
- `profil.html` — Mon profil
